E4. Preliminar version of the handbook (v1)

This practical guide aims to present the data spaces created in the INESData project as well as explain the creation process so that those interested in developing data spaces can have a practical reference. Given that the project is still in its early stages, this version of the guide will focus, mostly, on explaining what data spaces are and analyzing the different international initiatives that serve as a reference.

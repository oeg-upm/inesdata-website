E4. Preliminar version of the handbook (v1)

This practical guide aims to present the data spaces created in the INESData project as well as explain the creation process so that those interested in developing data spaces can have a practical reference. Given that the project is still in its early stages, this version of the guide will focus, mostly, on explaining what data spaces are and analyzing the different international initiatives that serve as a reference.

Link

E5. Horizontal components for data spaces

Comprehensive vision of the project that addresses its use cases, technological architecture, and AI services for data management. It details the cloud infrastructure, authentication mechanisms, interoperability, and auditing services, providing a solid framework for the efficient exchange of data between organizations.

Link (Spanish version)

E6. Handbook v2

Practical guide of the project that, in this edition, focuses on the governance of data spaces, incorporating three fundamental annexes:

  • Practical guide on compliance issues with Data Space regulations (A1).
  • Definition of roles in Data Space governance (A2).
  • Standard contractual models (A3).
Link (Spanish version)

E8. Initial version of the creation of vertical data spaces

Presentation of the initial progress in the creation and deployment of vertical data spaces on the INESData infrastructure. Each data space focuses on a specific vertical domain —mobility, digital media, linguistic, and legal—, integrating advanced services and artificial intelligence technologies to address the particular needs of each sector.

Link (Spanish version)

E10. Initial version of cloud and edge services for knowledge graph generation

Services for the creation of declarative rules and knowledge graphs, with a flexible and scalable tool that optimizes data usage, transforming and structuring it into a graph through intuitive and reproducible rules to facilitate analysis and decision-making.

Link (Spanish version)

E12. Initial version of natural language processing services

Microservices integrable into any data space requiring advanced capabilities for text processing (designed for the Spanish language).

  • Services for enriching language models with knowledge graphs.
  • Creation of a linguistic data space.
  • Data space and knowledge services for the legal sector and public procurement.
Link (Spanish version)

E14. Initial version of cloud services for large-scale multimodal and multimedia analysis

Description of the proposed multimodal microservices in the digital media data space, capable of transcribing, translating, anonymizing text, audio, and video, as well as performing various format conversions.

Link (Spanish version)
